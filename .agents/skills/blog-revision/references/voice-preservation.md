## なぜ地の文をリスペクトするのか

執筆者の音声入力や自然な語り口には、以下の価値がある。

- **個性と人間味**: 「〜なぁといったところです」「ちょっとやる意味ないのかなあ」といった口語的表現は、執筆者の個性を表す
- **感情と温度感**: 「あれ、頻度少ないし」「やっぱり偉大だなでしょう」といった素直な感想は、読者との距離を縮める
- **リズムと読みやすさ**: 自然な語り口は、堅苦しくない親しみやすい文章を生み出す

過度に整理・体裁を整えすぎると、これらの価値が失われ、機械的で冷たい文章になってしまう。

## 良い例と悪い例

### 悪い例(過度な整理)

```
元の文章:
「そもそもの話として、あまり記事を更新頻度高く載せられていないのに、スマホから記事を書くとか文章を構成するなんてことは、鶏が先か卵が先かみたいな問題でした。」

過度に整理した文章:
「記事の更新頻度自体が高くない状況で、スマホから記事を書いたり文章を構成したりすることは、鶏が先か卵が先かという問題でした。」

問題点:
- 「そもそもの話として」という導入の温かみが消失
- 「〜なんてことは」という口語的な強調が失われ、機械的になった
```

### 良い例(最小限の修正)

```
元の文章:
「そもそもの話として、あまり記事を更新頻度高く載せられていないのに、スマホから記事を書くとか文章を構成するなんてことは、鶏が先か卵が先かみたいな問題でした。」

最小限の修正:
「そもそもの話として、あまり記事を更新頻度高く載せられていないのに、スマホから記事を書くとか文章を構成するなんてことは、鶏が先か卵が先かみたいな問題でした。」

修正内容:
- 句読点の調整で読みやすさを微調整
- 温かみのある表現はそのまま保持
```

## 添削の具体的な指針

### 1. 最小限の修正

- 音声入力の明らかな誤字脱字のみ修正する
- 句読点や助詞の調整で読みやすさを向上させる
- 表現の言い換えは極力避ける

### 2. 口語的表現を保持する

- 「〜なぁといったところです」「〜ってところです」→ そのまま保持
- 「ちょっと〜」「やっぱり〜」「結構〜」→ 親しみやすさとして残す
- 「あれ、〜」「なんだかんだで〜」→ 語り口として尊重

### 3. 感情表現を大切にする

- 「〜でしょう」「〜と感じました」→ 個性として保持
- 反省や正直な感想(「まだちゃんと活用できている気がしませんが」)→ 人間味として残す
- 比喩表現(「鶏が先か卵が先か」)→ そのまま活用

### 4. 修正が必要な要素

- 音声入力の誤字(「痴漢」→「置換」、「九大点」→「及第点」)
- 意味が通じない助詞や接続詞
- 誤った漢字の使い方(「言う」→「いう」など文脈によって調整)
- 明らかな文法エラー

## 添削前のチェックリスト

音声入力や自然な文章を添削する前に、以下を確認する。

- [ ] この文章の個性や温かみを理解しているか?
- [ ] 整理することで失われる価値はないか?
- [ ] 最小限の修正で読みやすくできるか?
- [ ] 執筆者の意図を正確に汲み取っているか?
