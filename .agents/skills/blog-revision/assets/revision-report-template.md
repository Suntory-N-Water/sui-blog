# 推敲レポート出力テンプレート

blog-revision スキルが推敲モードで出力する形式。各セクションは「軸ごとに該当があれば書く、なければ『該当なし』」とする。書き直し全文は最後に 1 つだけ置く。

```md
## 推敲レポート

### 判定
- 論調: 硬めの論調 / ふわっとした論調
- 適用軸: アテンションライティング / 作文技術 / 両方

### アテンションライティング軸
- 冒頭診断: (所見)
- 1 文長: (60 字超の文の指摘)
- 離脱ポイント: (段落の継ぎ目で引っ掛かりが無い箇所)
- 無駄な装飾: (前置き・自己言及・冗長表現)

### 作文技術軸
- 修飾語の順序: (違反箇所とルール名)
- 読点: (打刻の過不足)
- 入れ子構造: (主語と述語が遠い箇所)
- 表記: (漢字・ひらがな・家畜語の指摘)

### 書き直し
(全文。元の論旨・固有名詞・主張は変えない)
```

## 出力時のルール

1. ユーザーが「全文だけ」「差分だけ」「指摘だけ」と指示した場合はそれに従う。
2. 強調表現の `**` は出力しない。
3. 該当が無い項目は省略せず「該当なし」と書く。
4. 「適用軸」がアテンションライティングのみの場合は作文技術軸のセクションを省略してよい。逆も同様。
